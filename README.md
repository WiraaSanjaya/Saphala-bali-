# Saphala Bali — Restaurant Website

A single-page, single-file website for **Saphala Bali**, a restaurant located in Kesiman, Denpasar Timur, Bali. Built with plain HTML, CSS, and JavaScript — no frameworks, no build step, no external dependencies. Just open `saphala-bali.html` in a browser.

## ✨ Features

- **Hero section** with the restaurant's interior photo as a background
- **About section** introducing Saphala Bali
- **Interactive menu** with tabbed categories:
  - Small Bites
  - Salad
  - Pasta
  - Burger & Sandwich
  - Main Dish
  - Steak, Pork & Specials
  - Sweet Treats
  - Beverages
- **Photo gallery** showcasing the venue (signage, terrace, dining room, evening ambiance, river-view deck)
- **Visit Us** section with address, contact info, opening hours, and an embedded Google Maps location
- Fully **responsive** layout (desktop, tablet, mobile) with a mobile hamburger nav
- All images are **embedded as Base64**, so the entire site is contained in **one HTML file** — easy to host or share

## 🎨 Design

- Color palette: forest green, warm beige, and terracotta — earthy tones reflecting a Balinese, local-food vibe
- Typography: [Fraunces](https://fonts.google.com/specimen/Fraunces) (serif, headings) + [Lato](https://fonts.google.com/specimen/Lato) (body text)

## 🚀 Usage

1. Download `saphala-bali.html`
2. Open it directly in any modern browser, **or**
3. Deploy it to any static hosting service (GitHub Pages, Netlify, Vercel, etc.)

### Deploying with GitHub Pages

1. Push `saphala-bali.html` to this repository
2. (Optional) Rename it to `index.html` so it loads automatically at the root URL
3. Go to **Settings → Pages**, select your branch and root folder, and save
4. Your site will be live at `https://<username>.github.io/<repo-name>/`
